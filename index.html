ONLINE SHOP



Design and implementation of a web application that will allow registered users to post ads for the sale of technology products. Registered users can therefore register as many ads as they want accompanied by photos. Visitors, as well as registered users of the website, can search based on a product category such as (laptops, desktops, phones, tablets) and the search results are displayed paginated. Each user has the possibility to modify or completely delete his ad later. In addition, the administrator (Admin) will be the only one who can create or modify a category. The system is implemented in the PHP programming language, which additionally uses the Laravel framework for the backend of the application, and phpMyAdmin is used for managing MySQL databases through a browser. On the other hand, Bootstrap has been used for the frontend as the display framework of the website.

Home Page

Admin Page

In the application there are two user roles for example simple user and administrator. The administrator (Admin) of the page can add a new category by pressing the Add New button, or edit it with the Edit button, or delete it by pressing the Delete button.

Register Page

During registration, a new user has the possibility to create a new account, filling in all the information requested.

Now, in case of incorrect information, the appropriate error messages are displayed.

Login Page

On this page, the user will be able to log in after first creating an account. By entering the “Username” and “Password” he can log into his account. Login is required for users who wish to post an ad.

Create Product

Each user who is registered in the application has the possibility, as long as he is connected, to register as many advertisements as he wishes. From the menu he selects entry and then chooses the product category he wishes to publish. It then fills in the information required by the application. You can also upload an image for the product. After he is finished he can press Submit to publish his product.

Product Page

The image below shows all products paginated by 6 and categorized by date of entry for all registered users or not.

User Product page

On this particular page, more details are displayed for each product separately by pressing the Product description button and the information of the product owner by pressing Contact Info.

Edit Product

On this page, the user can edit his data if he has made a mistake. The values in the form below are pre-filled based on the information entered by the user.

Delete Product

On this particular page, the user has the option to delete the product if it is his. A confirmation message is displayed before the product is deleted.

Contact Page

On this page the user can contact the page administrator through a form where he enters his first name, last name, email, mobile phone, subject and message. All these elements are mandatory and if they are filled in incorrectly, there will be error messages specific to each one.
